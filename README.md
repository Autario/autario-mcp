# autario-mcp

MCP server for [autario](https://autario.com) | Search, query, and publish charts across 2,500+ verified public datasets.

## Quick Start

```json
{
  "mcpServers": {
    "autario": {
      "command": "npx",
      "args": ["autario-mcp"]
    }
  }
}
```

Or install globally:

npm install -g autario-mcp
